SimpleRubyDiscordBot

A simple Discord bot written in ruby with submodules

  1. If not done already go to the Discord dev portal and create your bot app to get a token and client_id

  2. Add your data to the config file

  3. On Windows you can update your doc by using the .bat

  4. Install the necessary gems:

    • gem install discordrb

    • gem install wikipedia-client

    • gem install nokogiri

    • gem install reverse_markdown

  5. Make the bot run with the SimpleRubyDiscordBot.rb
